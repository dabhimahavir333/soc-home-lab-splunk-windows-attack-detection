SOC Home Lab – Splunk Windows Attack Detection

This project simulates real-world cyber attacks on a Windows system

and detects them using Splunk SIEM.

Lab Architecture:

Kali Linux (Attacker)

      ↓

Windows 10 (Victim)

      ↓

Sysmon Endpoint Telemetry

      ↓

Splunk Universal Forwarder

      ↓

Splunk SIEM

      ↓

Detection Queries \& Alerts



Technologies Used

\- Splunk Enterprise

\- Sysmon

\- Kali Linux

\- Windows 10

\- Nmap

\- PowerShell



Attack Simulations

\- Nmap Port Scanning

\- Suspicious PowerShell Execution



Detection Rules

\- Port Scan Detection

\- PowerShell Attack Detection

\- Process Execution Monitoring

\- Network Connection Monitoring



Outcome

This lab demonstrates SOC analyst skills including log analysis,

attack detection, and SIEM investigation.
